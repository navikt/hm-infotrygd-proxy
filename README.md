# hm-infotrygd-proxy

Applikasjon som snakker med replikasjonsdatabasen til Infotrygd og publiserer relevante endringer til hm-rapid.

## Teknologier

Prosjektet bruker Kotlin og Gradle, i tillegg til konseptet Rapids and Rivers.
